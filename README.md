# 浙江省初中知识点打卡刷题 - Web 版

## 项目概述
基于 Web 的初中刷题应用，支持浙江省 2026 年初中教材的每日打卡练习。

## 功能特性
- ✅ 支持七、八、九年级（上册/下册）
- ✅ 5 大科目：语文、数学、英语、科学、社会
- ✅ 每科 100 题题库，每天每科随机抽 10 题
- ✅ 每日 50 题（5 科×10 题）混合练习
- ✅ 实时判断对错
- ✅ 详细答案解析
- ✅ 错题本自动保存
- ✅ 历史记录追踪
- ✅ 打卡进度统计
- ✅ 响应式设计，手机/平板/电脑通用

## 技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- localStorage 本地存储
- 无需后端，纯前端实现

## 使用方式
1. 直接用浏览器打开 `index.html`
2. 或部署到任何 Web 服务器
3. 手机可添加到主屏幕，像 APP 一样使用
